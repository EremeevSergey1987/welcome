Системы контроля версий позволяют сохранять историю изменений программного кода и работать над проектами в команде.

cd /var/www/html/welcome

git pull - затянуть (взять) изменения с сервера
-------------Команды для отправки изменений в Git-----------------
git status - просмотреть перечень сделанных нами изменений
(Команда git status позволяет просмотреть состояние рабочей директории — перечень незакоммиченных изменений.)

git add . - с помощю этой команды мы добавляем изменения в индекс
git commit -m 'work_19' - Создаем коммит с описанием изменений
git push - отправить коммит в удаленный репозиторий
------------------------------------------------------------------

git show - Для просмотра последнего коммита можно использовать как команду git show, так и команду git log, выводящую всю историю коммитов.
git log - выводит всю историю коммитов.
